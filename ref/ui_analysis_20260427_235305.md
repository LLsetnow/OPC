# UI 界面详细分析报告

## 1. 布局结构

该界面采用 **非对称网格自由布局（Asymmetric Grid Layout）**，属于现代 Glassmorphism 风格的个人仪表盘（Personal Dashboard）。整体结构可划分为以下区域：

| 区域 | 描述 | 坐标范围（估算） |
|------|------|------------------|
| **顶部区域** | 包含问候语卡片与实时时钟 | Y: 80–300 |
| **主内容区（左）** | 日历组件 + GitHub 快捷入口 | X: 160–420, Y: 370–870 |
| **主内容区（中）** | 音乐播放器 + 图片画廊 | X: 450–730, Y: 390–910 |
| **侧边栏（右）** | 垂直导航菜单 | X: 1160–1390, Y: 340–700 |
| **背景层** | 渐变底色 + 噪点纹理 | 全屏覆盖 |

**布局特点**：
- 无传统顶部导航栏，采用 **浮动卡片式（Floating Cards）** 设计
- 使用 CSS Grid 或绝对定位实现错落有致的排版
- 各模块间距不均等，营造动态视觉层次感

## 2. 组件清单

### 2.1 问候卡片（Greeting Card）
- **类型**：信息展示卡片
- **内容**：
  - 主标题："Hi, I'm [Name]"（字号 ~32px，字体权重 600）
  - 装饰元素：文字下方短横线（宽 40px，高 3px，圆角）
  - 操作按钮：右上角三点菜单图标（`...`）
- **样式**：毛玻璃背景，圆角 24px，内阴影

### 2.2 模拟时钟（Analog Clock Widget）
- **类型**：时间显示组件
- **内容**：
  - 时间："10:24"（数字字体，字号 ~42px）
  - 日期："May 20, 2024"（字号 ~12px，灰色）
  - 刻度盘：60 个细小刻度点围成圆形
- **样式**：白色半透明圆形容器，细边框（1px solid rgba(255,255,255,0.8)）

### 2.3 日历组件（Calendar Widget）
- **类型**：日期选择器/展示器
- **内容**：
  - 头部：左右箭头（< >）+ "May 2024"
  - 星期行：MON-SUN（英文缩写，字号 11px，灰色）
  - 日期网格：6×7 矩阵（1-31 号）
  - 当前选中：**20** 号（蓝色实心圆背景，白字）
- **交互状态**：悬停时其他日期可能有微弱高亮

### 2.4 音乐播放器（Music Player Card）
- **类型**：媒体控制卡片
- **内容**：
  - 左侧图标：音符符号（圆形浅蓝背景，尺寸 48×48px）
  - 曲目信息："Dream It Possible"（歌名，14px）+ "Delacey"（艺术家名，12px，灰色）
  - 进度条：线性进度条（已播放部分蓝色，总长浅灰）
  - 时间戳：左 "02:35"，右 "03:24"
- **样式**：圆角 20px，内部 padding 20px

### 2.5 导航菜单（Navigation Menu）
- **类型**：垂直列表导航
- **内容**（4 项）：
  1. 📁 我的项目（My Projects）
  2. 👤 关于网站（About Website）
  3. 📰 每日新闻（Daily News）
  4. ⭐ 收藏夹（Favorites）
- **样式**：每项高度 ~56px，图标与文字水平排列，hover 有背景色变化

### 2.6 GitHub 链接卡（External Link Card）
- **类型**：快捷操作按钮/卡片
- **内容**：
  - 左侧：GitHub Octocat 图标（深灰圆形背景，32×32px）
  - 文字："GitHub"（16px，中等字重）
  - 右侧：箭头图标（→）
- **样式**：胶囊形/圆角矩形，hover 时可能右移或变色

### 2.7 图片画廊（Image Gallery Grid）
- **类型**：媒体网格展示
- **内容**：2 行 × 4 列 = 8 张缩略图
  - 图片内容：雪山、海面、海洋、晚霞、花朵、几何图形、纯色块、建筑
- **样式**：每张图圆角 12px，间距 12px，图片对象-fit: cover

## 4. 交互元素

| 元素 | 交互类型 | 预期行为 |
|------|----------|----------|
| **三点菜单（...）** | 点击 | 弹出下拉菜单（设置/编辑/更多选项） |
| **日历箭头（< >）** | 点击 | 切换上/下月份，带过渡动画 |
| **日历日期** | 点击 | 选中日期，更新下方事件列表或高亮状态 |
| **音乐进度条** | 拖拽/点击 | 跳转播放进度 |
| **音乐封面/名称** | 点击 | 打开完整播放器或暂停/播放切换 |
| **导航菜单项** | Hover | 背景色变为 `rgba(129, 140, 248, 0.1)`，左移 4px |
| **GitHub 卡片** | Click/Hover | Hover 时箭头右移 4px；Click 打开新标签页 |
| **画廊图片** | 点击 | 打开 Lightbox 全屏查看或进入详情页 |
| **时钟** | 静态展示 | 实时更新（每秒），刻度盘可能随秒针旋转 |

## 5. 数据结构

### 5.1 用户信息模型（User Profile）
```typescript
interface UserProfile {
  displayName: string;        // "[Name]"
  greetingText: string;       // "Hi, I'm "
  avatar?: string;            // 可选头像 URL
}
```

### 5.2 日历数据模型（Calendar State）
```typescript
interface CalendarData {
  currentYear: number;        // 2024
  currentMonth: number;       // 5 (May)
  selectedDate: number;       // 20
  todayDate: number;          // 20
  events?: CalendarEvent[];   // 可选的事件标记
}
```

### 5.3 音乐播放器模型（Now Playing）
```typescript
interface TrackInfo {
  title: string;              // "Dream It Possible"
  artist: string;             // "Delacey"
  albumArt?: string;          // 封面 URL（当前使用默认图标）
  duration: number;           // 204 秒 (03:24)
  currentTime: number;        // 155 秒 (02:35)
  isPlaying: boolean;
}
```

### 5.4 导航菜单模型（Menu Items）
```typescript
interface MenuItem {
  id: string;
  label: string;              // 中文标签
  icon: string;               // Icon 名称或 SVG 路径
  route: string;              // 路由路径
  badge?: number;             // 可选通知数
}
```

### 5.5 画廊数据模型（Gallery Items）
```typescript
interface GalleryItem {
  id: number;
  src: string;                // 图片 URL
  alt: string;                // 描述文本
  category?: string;          // 分类（风景/抽象等）
}
```
